More Data At Once
The library serves query results that are optimized for data-immediacy, exposing all data directly within the results page. Without the need for a full-details page, this eliminates back-and-forth page traversal. The exposure of all records provides an opportunity for curiosity to glance over at other records meangingfully. Every dataset so far benefits from this manner of data presentation. Long text is collapsed into description containers with show-more toggles. The results of the data appear as reasonable article space, usually containing a thumbnail of 300 pixel width to the left with meta-data printed to the right in a two panel element which either occupies its own row or is limited in width to act as a card which can hold a few per line depending on the user's chosen application display width.
Pagination

Every dataset uses pagination. The page limit is curated for each implementation based on the average html output length of a query. Art museums mostly show 20 records per query due to their meta-data and thumbnail richness. Implementations with less meta-data output will show a higher number of records. Pagination creates tunable results, which resolves any issues of data over-abundance. Deep paging inefficiencies are addressed with a hybrid method method which uses both keyset and offset pagination.

Creator Representation

When any opportunity arises, creators of contemporary records are represented directly in the presentation of data. Their user-names and profile pages are outputted to accompany any applicable source page that links to the original record. Giphy provides this opportunity because they accepted Internet Public Library as a verified application, which lifts their rate limiting while also not restricting application design. Every other image hosting API such as Unsplash and Pixabay do not allow the creation of applications which resemble media searches. This prevents access to their API entirely, which also prevents the ability to do what Giphy allows, which is represent contemporary artists through the meta-data. The ability to represent a contemporary creator in such a way is rare for this reason.
