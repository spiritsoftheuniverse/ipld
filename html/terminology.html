A few terms are informally used frequently in this documentation:

"The library" Internet Public Library.
Dataset A collection of records that is usually gathered from an external source and stored into a database for reading.
Data Record(s) from a dataset.
Implementation A dataset that is fully functional and presented in the library.
"Unique" Used in the case of unique implementation, where unique is of bespoke data structure. This is in contrast to Mediawiki datasets which are declared by the framework's table creation script.
Filter A user-interface element commonly declared by button or title, that is used as toggle data for queries.
Category-Link A user-interface element commonly declared by title, that is used to trigger category browsing queries.
