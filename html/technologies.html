This is a list of all the technologies and code that make the library run.

Name Purpose
w3 HTML, CSS, Javascript, MySQL, PHP. Process and display data.
jQuery Document traversal.
Web Audio API Process web audio with timing precision and playback details.
IndexedDB API Running a local database for user-cache.
localStorage User-interface cursor memory and other state management.
Fancybox Lightbox image gallery viewer.
OpenSeadragon Tile-zooming system for large images.
tonal Music theory library, integrated into the Music Theory application.
Composer PHP dependency manager.
Mediawiki Manages all states and queries for Mediawiki datasets.
guzzle Streaming Nasa dataset synchronization.
.env Separating secrets from code.
Git Source control.
